# MoH eLMIS Partner Mapping Dashboard

Interactive mockup for mapping Ministry of Health eLMIS partners by geography, eLMIS role, support type, funding execution, agreement status, compliance, and focal person.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Included views

- Province partner-density heatmap
- eLMIS module and RACI-style role matrix
- Committed vs disbursed funding bars
- Renewal and compliance risk register
- Searchable partner directory with support type and status filters
